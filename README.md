# 📖 About

Charlos is an advanced multipurpose Discord bot designed to provide everything your server needs in a single package.

Whether you're managing a large community, playing economy games with friends, searching for information, or simply having fun, Charlos offers an extensive collection of commands built with speed, reliability, and ease of use in mind.

The project is actively developed with new commands, improvements, and quality-of-life updates released regularly.
